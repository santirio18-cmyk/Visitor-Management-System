# Fix: Clear Browser Cache

## The Problem
Your browser is showing "your-backend-url" because it's using a **cached old version** of the JavaScript file.

## The Solution (Do This Now)

### Step 1: Hard Refresh Your Browser
**Chrome/Edge (Windows):** Press `Ctrl + Shift + R` or `Ctrl + F5`
**Chrome/Edge (Mac):** Press `Cmd + Shift + R`
**Firefox:** Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
**Safari:** Press `Cmd + Option + R`

### Step 2: Clear Cache (If Hard Refresh Doesn't Work)
1. Open Developer Tools: Press `F12` or `Right-click → Inspect`
2. Right-click the refresh button (next to address bar)
3. Select **"Empty Cache and Hard Reload"**
