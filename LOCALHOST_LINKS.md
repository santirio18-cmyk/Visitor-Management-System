1. Press `F12` to open Developer Tools
2. Go to **Console** tab
3. Look for any red errors

```bash
# Stop current servers (Ctrl+C in terminal)
# Then restart:
npm run dev
```

### Step 3: Clear Browser Cache
- Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
