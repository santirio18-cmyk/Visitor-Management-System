## How to Start

### Step 1: Install Dependencies (First Time Only)
```bash
npm run install-all
```

### Step 2: Start Both Frontend and Backend
```bash
npm run dev
```

This will start:
- ‚úÖ Frontend at: **http://localhost:3001**
- ‚úÖ Backend at: **http://localhost:5001**

---

## Or Start Separately

### Start Backend Only:
```bash
npm run server
```
Backend runs at: **http://localhost:5001**

### Start Frontend Only:
```bash
npm run client
```
Frontend runs at: **http://localhost:3001**
