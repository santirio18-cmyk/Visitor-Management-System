## Step 1: Create Repository on GitHub
1. Go to https://github.com
2. Click "+" ‚Üí "New repository"
3. Name it: `vendor-management-system`
4. Choose **Public**
5. **DO NOT** check "Add a README file" (we already have one)
6. Click "Create repository"

## Step 2: Upload the ZIP File
1. On the new repository page, you'll see a gray box with options
2. Click the link: **"uploading an existing file"** (usually in that gray box)
3. OR click **"Add file"** button (top right) ‚Üí **"Upload files"**
4. Drag and drop the file: **`vendor-management-system.zip`** onto the page
5. Scroll down to the bottom
6. Enter commit message: "Initial commit: Vendor Management System"
7. Click **"Commit changes"** button

**That's it! Done!** ‚úÖ
